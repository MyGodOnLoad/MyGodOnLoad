报错

1209 - The MySQL server is running with the–read-only option so it cannot execute this statement

一般这个错误有两种原因：

  1. 连到从库了。从库一般设置为只读。

  2. 主库的read_only参数被修改为1

暂时解决

查看mysql配置

永久解决

因为写入数据量太大, 内存溢出导致了mysql的重启

配置文件中设置了read_only ，所以重启会导致read_only=1
